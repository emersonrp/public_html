My Resume

Some tiny pieces of software I've written or worked on:

setaffin -- a little wrapper program for Windows NT/2000/XP(?) to set the CPU affinity of a process on a multi-CPU box. If you have Windows 95/98/ME, or if you have only one CPU/core in your machine, this will not do anything for you.

syntax.tcl -- a plugin for the cross-platform chat client tkMOO-light that enables syntax highlighing for MOOcode in the local edit window.

camwatch -- a -very- lightweight program for Unix / X for watching/archiving webcams. Very much a work in progress. This program was appropriated from its original author, cblack, who seems to have disappeared off the face of the earth.
